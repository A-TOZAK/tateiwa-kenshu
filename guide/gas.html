Google Apps Script

GASでスライドを量産する

「同じ形のスライドをたくさん」は、手で作る仕事ではありません。スプレッドシートに言葉を並べれば、スライドは自動でできます。プログラミングの知識は不要、コピペで動きます。

What for

なにが作れるか

  • 漢字・英単語・九九のフラッシュカード（1語1枚・でか文字）
  • 音読カード・都道府県カードなど、暗記もの全般
  • 名簿から1人1枚の賞状・係カード・めあてカード
目安：漢字50語のフラッシュカードを手で作ると1時間仕事。この方法なら、シートに打つ10分＋実行1分です。浮いた50分は2学期のゆとりにどうぞ。
Recipe 1

フラッシュカードを自動で作る

  1. スプレッドシートを新規作成。A列に出す言葉、B列に読みや答え（B列は空でもOK）を並べる
  2. メニューの 拡張機能 → Apps Script を開く
  3. 最初から入っている数行を全部消して、下のコードを貼り付ける
  4. 上の 実行（▶）を押す。初回だけ「承認が必要です」と出るので、自分のアカウントを選んで許可する（自分で貼ったスクリプトを自分に許可する画面です）
  5. ドライブに「（シート名）フラッシュカード」というスライドができています
コード（全部コピーして貼り付け）
function makeFlashcards() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  const rows = sheet.getDataRange().getValues();
  const deck = SlidesApp.create(sheet.getName() + ' フラッシュカード');
  deck.getSlides()[0].remove(); // 最初の空スライドを消す

  rows.forEach(function(row) {
    const omote = String(row[0] || '').trim(); // A列＝出す言葉
    const ura   = String(row[1] || '').trim(); // B列＝読み・答え
    if (!omote) return;
    addCard(deck, omote);
    if (ura) addCard(deck, ura);
  });
}

function addCard(deck, word) {
  const slide = deck.appendSlide(SlidesApp.PredefinedLayout.BLANK);
  const box = slide.insertTextBox(word, 0, 0, 720, 405); // 画面いっぱいの箱
  box.getText().getTextStyle()
     .setFontSize(word.length > 6 ? 90 : 140) // 長い言葉は少し小さく
     .setBold(true);
  box.getText().getParagraphs().forEach(function(p) {
    p.getRange().getParagraphStyle()
     .setParagraphAlignment(SlidesApp.ParagraphAlignment.CENTER);
  });
  box.setContentAlignment(SlidesApp.ContentAlignment.MIDDLE);
}
しくみ：A列の言葉→B列の答え→次の行、の順にスライドが並びます。「漢字→読み」「英単語→意味」「6×7→42」。シートを差し替えれば何度でも使えて、学年の先生にシートごと渡せます。
Recipe 2

名簿から賞状・カードを1人1枚つくる

賞状のテンプレスライドを1枚作り、名前の場所に {{名前}} と打っておきます。あとは名簿のシートからこのコードを実行すると、全員分のページができます。

  1. Googleスライドで賞状を1枚デザインし、名前の位置に {{名前}} と書く
  2. そのスライドのURLの /d/ と /edit の間の長い文字列（ID）をコピーする
  3. 名簿のスプレッドシート（A2から下に名前）で 拡張機能 → Apps Script を開き、下のコードのIDを差し替えて実行
コード（IDを1か所差し替え）
function makeCertificates() {
  const TEMPLATE_ID = 'ここにテンプレのスライドIDを貼る';
  const names = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet()
    .getRange('A2:A').getValues().flat().filter(String);

  const copy = DriveApp.getFileById(TEMPLATE_ID).makeCopy('賞状（全員分）');
  const deck = SlidesApp.openById(copy.getId());
  const master = deck.getSlides()[0];

  names.forEach(function(name) {
    master.duplicate().replaceAllText('{{名前}}', name);
  });
  master.remove(); // ひな形のページを消す
}
応用：{{名前}} 以外にも {{がんばったこと}} {{日づけ}} など、列を増やせば差しこむ言葉を増やせます。まずは名前だけで1回動かしてから広げるのがおすすめです。
Notes

つまずきやすいところ

「拡張機能」にApps Scriptが出てきません。
学校アカウントの設定で止められている場合があります。その場合は無理をせず、情報担当かICT支援員さんに「Apps Scriptは使えますか」と聞いてください。個人の研究用アカウントで試す手もあります。
初回の承認画面がこわいです。
「このアプリは確認されていません」という画面は、自分で貼ったスクリプトに自分が許可を出すときの定型の画面です。作った本人が実行する分には進めて大丈夫です（学校のICT利用ルールがある場合はそちらに従ってください）。
エラーで止まりました。
いちばん多いのはコードの貼り忘れ（一部だけ貼る）です。いったん全部消して、全部貼り直してください。それでも動かなければ、エラー文をそのまま公式LINEに送ってもらえれば見ます。
